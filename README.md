# Yeoman Bootstrap Project Generator 

## Getting Started

#### 1. Install Yeoman

```bash
npm install -g yo
``` 

#### 2. Install bootstrap project generator

To install generator-bs from npm, run:

```bash
npm install -g generator-bs
```

Finally, initiate the generator:

```bash
mkdir project_dir && cd $_
yo bs
```

Start development.

```bash
grunt dev
```
